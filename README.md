# PSTriageScripts

### A. Required Binaries

1. In order for this script to run properly, you will need to download the following executable files directly from their main download site and place them in a folder named "\bin" at the root of directory in which the `PowerShell_DFIR_Script.ps1` is located:

    - `sqlite3.exe` (part of the `sqlite-tools-win-x64-3530100.zip` download) from [SQLite Home Page](https://sqlite.org/)
    - `PSInfo.exe` (part of the `SysInternals Suite`) from [PsInfo - Sysinternals | Microsoft Learn](https://learn.microsoft.com/en-us/sysinternals/downloads/psinfo)
    - `MagnetRAMCapture.exe` from [Magnet RAM Capture](https://www.magnetforensics.com/resources/magnet-ram-capture/)
    - `MagnetProcessCapture.exe` from [Magnet Process Capture](https://www.magnetforensics.com/resources/magnet-process-capture/)
